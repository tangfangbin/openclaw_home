# 图像合成技能 - 学习报告

## 研究发现

### 问题分析
之前的效果差是因为：直接将JPG图片叠加，没有进行**背景移除**，导致：
- 原图的复杂背景残留
- 光照/色调不匹配
- 边缘生硬

### 正确的图像合成流程

```
原始图片 → 背景移除(抠图) → 获得透明PNG → 光照调整 → 合成到新背景
```

### 最佳方案对比

| 方案 | 优点 | 缺点 |
|------|------|------|
| **rembg** (本地Python库) | 免费、可离线 | 安装复杂、依赖多 |
| **remove.bg API** | 效果好、专业 | 需要API Key、有成本 |
| **Photoroom** | 效果好、有免费额度 | 需要注册 |
| **Clipdrop** | API稳定、有免费额度 | 需要API Key |
| **Canva可画** | 操作简单 | 需要手动操作 |

### 推荐工作流

**方案A：使用rembg本地抠图（需要解决安装问题）**
```bash
pip3 install rembg
python3 -c "from rembg import remove; remove('input.png', 'output.png')"
```

**方案B：使用Photoroom/Canva手动处理**
1. 上传到 Photoroom.com
2. 自动移除背景
3. 下载透明PNG
4. 再进行合成

**方案C：使用MiniMax图像生成进行参考**
- 不能完美保持原图，但可以生成相似风格的新图

## 技能现状

### 当前能力
- ✅ 基础图像叠加（PIL直接paste）
- ✅ 添加文字水印
- ✅ 尺寸调整

### 当前局限
- ❌ 无法自动抠图（rembg安装失败）
- ❌ 无法光照/色调匹配
- ❌ 复杂边缘处理差

### 需要人工配合
1. 用户需要提供**分开的**前景图和背景图
2. 前景图最好是**透明背景PNG**
3. 或者用户先用Photoroom/remove.bg手动抠图
